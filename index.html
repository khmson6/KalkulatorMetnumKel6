📖 USER MANUAL (Panduan Pengoperasian Aplikasi)

1 Input Titik Koordinat

Masukkan koordinat data pada kolom Nilai X dan Nilai Y. Gunakan tombol + Tambah Baris Data untuk menambah titik, atau Hapus jika ingin mengurangi baris.

2 Target & Metode

Pilih salah satu metode interpolasi yang diinginkan. Masukkan nilai Titik Target (x*) yang ingin dicari nilai pendekatannya, serta Nilai Sejati f(x*) (jika ada) untuk memunculkan nilai Galat Mutlak.

3 Hasil & Analisis

Klik tombol HITUNG NILAI. Sistem akan menampilkan hasil interpolasi, matriks tabel proses selisih, rincian runtunan rumus langkah kerja, dan visualisasi grafik plot fungsi.

*Catatan Khusus: Metode Newton-Gregory (Maju/Mundur) memerlukan spasi antar koordinat X yang seragam (h konstan) agar perhitungan dapat dieksekusi secara valid oleh sistem.
